# Security Policy

## Supported Versions

This project currently supports the latest code on the default branch.

## Reporting a Vulnerability

Please report security issues privately to `317507360@qq.com` instead of
opening a public GitHub issue. Include:

- A short description of the issue.
- Steps to reproduce it.
- Affected files or commands.
- Any relevant logs, without credentials or private machine data.

You may also use GitHub Security Advisories if they are enabled for the
repository.

## Repository Security Recommendations

After pushing to GitHub, enable these repository settings where available:

- Secret scanning.
- Push protection.
- Dependabot alerts.
- Code scanning with GitHub Actions.
